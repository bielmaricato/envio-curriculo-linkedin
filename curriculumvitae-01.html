Sobre

Como Analista de Sistemas Sênior concentro-me em desenvolver soluções eficientes que atenda as necessidades do cliente.

Busco criar interfaces dinâmicas e interativas utilizando o framework Angular, além de construir aplicações robustas usando o framework Microsoft .NET, tanto para aplicações Windows (Windows Forms) como Web (Aspnet, .Net Core para desenvolvimento de APIs).

Entrega de soluções automatizadas: Robotic Process Automation (RPA), também conhecida como robótica de software, usa tecnologias de automação inteligente para executar tarefas repetitivas de escritório de trabalhadores humanos, como extrair dados, preencher formulários, mover arquivos e muito mais, melhorando processos e entregando eficiência operacional.

Utilizando as ferramentas Selenium para extração de dados em páginas web, tais como Correios, Banco Central do Brasil, OCR (Google) para leitura de arquivos, WinAutomation, UiPath e também extrações para ETL.

No cenário atual, onde a agilidade e a integração de sistemas são cruciais, aplico meus conhecimentos avançados em .Net Core, C#, Docker, AWS, Git e Web Services para otimizar sistemas existentes e implementar APIs RESTful com ASP.NET Web API.

A minha meta é sempre entregar software de alta qualidade que agregue valor real aos clientes e usuários finais.

Trabalhando com equipes Squads com metodologia ágil SCRUM, promovo um ambiente colaborativo que favorece a inovação e a excelência técnica.
